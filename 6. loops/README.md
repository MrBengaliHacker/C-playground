## ðŸ’» Program: Check Prime Number

**ðŸ“” Problem**: Write a program to check whether a given number is a prime number or not.

**ðŸ’¡ Solution**: [View Code](prime_number.c)

```c
#include <stdio.h>

int main()
{
  int n, isPrime = 1;

  printf("Enter the number: ");
  scanf("%d", &n);

  // Prime numbers are greater than 1
  if (n <= 1)
  {
    isPrime = 0;
  }
  else
  {
    for (int i = 2; i < n; i++)
    {
      if (n % i == 0)
      {
        isPrime = 0; // Found a divisor, not prime
        break;
      }
    }
  }

  if (isPrime)
  {
    printf("%d is a Prime Number.\n", n);
  }
  else
  {
    printf("%d is not a Prime Number.\n", n);
  }

  return 0;
}

```
### OUTPUT

```
Enter the number: 7
7 is a Prime Number.
```
```
Enter the number: 10
10 is not a Prime Number.
```

**ðŸ’¡ Solution**: [View Code](greatest_common_divisor.c)

```c
#include <stdio.h>

int main()
{
    int num1, num2, i, gcd;

    printf("Enter The First Number: ");
    scanf("%d", &num1);
    printf("Enter The Second Number: ");
    scanf("%d", &num2);

    for (i = 1; i <= num1 && i <= num2; i++)
    {
        if (num1 % i == 0 && num2 % i == 0)
        {
            gcd = i;
        }
    }

    printf("The Greatest Common Divisor Of %d And %d Is : %d", num1, num2, gcd);

    return 0;
}

```
